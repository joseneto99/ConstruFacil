# Projeto "ConstruFacil - Engenharia de software"

## a) Descrição textual dos sistema

O sistema ConstruFacil é uma plataforma online destinada a facilitar a conexão entre trabalhadores da construção civil e pessoas ou empresas que necessitam de serviços relacionados à construção e reformas. A plataforma busca simplificar o processo de contratação de profissionais, oferecendo um ambiente seguro e confiável para ambas as partes.

### Principais funcionalidades

1.  Cadastro de Usuários: O profissional pode fazer o cadastro com informações pessoais, habilidades e experiências e os usuarios podem também fazer o cadastro para busca e contratação de serviços.
2.  Busca de servirços: O sistema de busca profissionais com base em critérios específicos.
3.  Perfil de Profissionais: Tem informações Detalhadas do perfis dos trabalhadores com habilidades, experiências e avaliações, sendo eles feedback de clientes para construir reputação.
4.  Sistema de Contratação: Onde o usuario pode fazer a solicitação de orçamentos, pedindo o orçamentos diretamente pela plataforma, e tambem pode fazer a contratação segura e pagamento integrado do serviço.
